# Maison Or — Premium E-Commerce (Marble & Gold)

A high-performance e-commerce prototype built with TanStack Start, React 19, Tailwind CSS v4, and Framer Motion.

## Getting Started (npm)

```bash
npm install
npm run dev
```

Then open the URL printed in the terminal (usually http://localhost:3000).

## Build

```bash
npm run build
npm run start
```

## Tech Stack
- TanStack Start v1 (React 19 + Vite 7)
- Tailwind CSS v4
- Framer Motion
- localStorage-based cart / wishlist / theme

All product data is mocked in `src/data/products.ts`.
